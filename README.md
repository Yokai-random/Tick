# Tick

一个简洁、好用的第三方 Minecraft 启动器。

为日本及海外网络环境下载游戏文件做了优化,支持正版微软登录、
多版本管理、模组与整合包一键导入。界面干净,开箱即用,免安装。

## 特性

- **正版微软登录** — 显示皮肤头像
- **全加载器支持** — Vanilla / Fabric / Forge / NeoForge
- **多实例管理** — 多个版本独立共存,互不干扰
- **资源下载** — 模组、光影、资源包(支持中文搜索)
- **整合包导入 / 导出** — 一键迁移你的整合包
- **服务器管理** — 内置服务端创建与控制台
- **Java 自动管理** — 自动下载匹配的 Java,无需手动配置
- **便携版** — 免安装,数据随程序目录走
- **浅色 / 深色主题**

## 下载

前往 [Releases](../../releases) 下载最新版本。

下载 `Tick x.x.x.exe`,双击运行即可,**无需安装**。
游戏数据(`.minecraft` 等)会保存在 exe 同目录下。

## 使用说明

1. 首次启动后,用微软账号登录。
2. 在「下载」页选择并安装你想要的版本。
3. 运行重度整合包前,请到「设置 → 内存」把内存上限调高
   (建议 **8 GB 以上**),否则可能因内存不足而崩溃。

## 系统要求

- Windows 10 / 11(64 位)
- 一个合法的 Minecraft 正版账号

## 构建(面向开发者)

```bash
npm install      # 安装依赖
npm run dev      # 开发模式运行
npm run build    # 打包为便携 exe(输出到 release/)
```
